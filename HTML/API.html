O Que é API?

API significa Application Programming Interface (em português, Interface de Programação de Aplicações). É um conjunto de comandos, protocolos e ferramentas que permite que dois sistemas ou programas se comuniquem. Ela define como os desenvolvedores devem escrever códigos para interagir com outro sistema, sem precisar saber como ele funciona internamente.


Exemplos Mais Práticos

Exemplo 1:

Imagine que uma API é como a interface de um controle remoto:

  • Você aperta um botão (faz uma chamada para a API).
  • A televisão responde (executa uma função).
  • Você não precisa saber como a TV funciona por dentro — só usar o controle corretamente.

Exemplo 2:

Situação: Um app de delivery mostra mapas e localização.

Ele não cria um mapa DO ZERO

Ele usa uma API do Google Maps, que fornece:

  • Mapa da região;
  • Localização do Usuário;
  • Rota entre dois pontos.

Tudo isso é obtido enviando pedidos para a API e recebendo resposta com os dados.
